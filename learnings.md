# 📚 Project Learnings

This project started with a simple goal: building a basic CRUD application using FastAPI. However, as I progressed, I realized I could make something much more practical and challenging. Instead of stopping at CRUD operations, I transformed the project into a **fully functional AI-powered News Application** with user authentication, a scalable backend architecture, MongoDB integration, and multiple NLP-powered AI features.

The project evolved from a beginner backend application into a production-style application that combines **FastAPI, MongoDB, Authentication, Hugging Face Transformers, and Streamlit** into one complete system. This journey taught me much more than I originally expected and helped me understand how real-world backend applications are designed.
